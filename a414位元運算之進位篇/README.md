* 內容：  
	一個數在電腦裡遞增時需要進位幾次。

* 輸入說明：  
	輸入的每一行有一個十進制正整數 N (1<=N<=2147483647)。輸入的最後一行有一個 0，代表輸入的結束，這個數字請勿做任何處理。
* 輸出說明：  
	對於每個正整數 N ，請輸出以二進制計算 N+1 時所需的進位次數。
