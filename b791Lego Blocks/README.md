* 內容：  
	你有兩種不同形狀的積木，分別為  
	  
	口口 （1*2）  
	和  
	口  
	口口口 (2*3的L型)  
	  
	這兩種。  
	  
	今天你想要使用這兩種積木來拼成 2*N 的形狀，請問共有幾種拼法？  

* 輸入說明：  
	本題為重複輸入多筆測資，以EOF結束。  
	每筆測資有一行，包含一個數字 N，1<=N<=25。
* 輸出說明：  
	輸出一行，包含一個數字，代表要拼成 2*N 的積木共有幾種拼法。
