##[a121：質數又來囉](http://zerojudge.tw/ShowProblem?problemid=a121)
======
評分方式：Tolerant 
記憶體限制：512 MB

1. 公開 測資點 (100%): 3.0s , <1M

題目最後更新時間：2011-05-06 18:43 

- - -
* 內容：  
	你的好朋友質數先生又來找你囉，給你兩個數字，請算出這兩個數字包含的範圍內有幾個質數。

* 輸入說明：  
	輸入兩個正整數a,b(1<=a<=b<=100000000)。  
	保證b-a <= 1000。  
* 輸出說明：  
	輸出一個非負整數，代表a到b之間(包含a,b)總共有幾個質數。
