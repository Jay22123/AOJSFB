##[a248：新手訓練 ~ 陣列應用](http://zerojudge.tw/ShowProblem?problemid=a248)
======

記憶體限制：512 MB

2. 公開 測資點 (95%): 1.0s , <1M

* 內容：  
	大家都知道，小算盤的小數運算只能算出小數點後三十幾位，但好奇的桑葉想知道更精準的小數值，請你幫可憐的桑葉做出可以算出精準的小數運算的程式。

* 輸入說明：  
	每次輸入有三個正整數 a , b , N，   
	(1<= a), (b <= 2147483647), (1 <= N <= 10000)，  
	輸入不會超過1000筆  。  

* 輸出說明：  
	請輸出 a / b 的小數運算結果，精準到小數點後N位，第N位以後請無條件捨去。
