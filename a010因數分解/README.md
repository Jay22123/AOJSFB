##[a010：因數分解](http://zerojudge.tw/ShowProblem?problemid=a010)
======

記憶體限制：512 MB

1. 公開 測資點 (100%): 1.0s , < 1K

* 內容：  
	各位在國小時都學過因數分解，都瞭解怎麼樣用紙筆計算出結果，現在由你來敎電腦做因數分解。  
	因數分解就是把一個數字，切分為數個質數的乘積，如 12 = 2^2 * 3  
	其中, 次方的符號以 ^ 來表示。

* 輸入說明：  
	一個整數, 大於1 且 小於等於 1000000。
* 輸出說明：  
	一個字串。
