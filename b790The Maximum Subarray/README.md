* 內容：  
	給一個長度為 N 的數列，請求出  
	1. 連續 subarray 的最大和
	2. 非連續（可以連著也可以跳著） subarray 的最大和。

	subarray 的長度不能為 0。

* 輸入說明：  
	第一行為一個數字 T，1<=T<=10，表示接下來有幾筆測資。  
	接下來每筆測資的第一行有一個整數 N，1<=N<=100000，代表數列長度。  
	接下來一行有 N 個整數，代表數列內容，-10000<=a[i]<=10000。
* 輸出說明：  
	輸出一行，包含兩個整數，用一個空白隔開。  
	第一個代表數列的連續子序列的最大可能和，  
	第二個代表數列的非連續子序列的最大可能和。

- - -
提示：  
	1. 非連續的意思是不一定要全部連續，可以連著也可以跳著。
	2. 子序列的長度不可以為 0，至少要挑一個數字。
